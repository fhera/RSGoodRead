EJERCICIO DE DJANGO

Se pide construir una web con un menú principal, que permita las siguientes opciones:

1. Dado un usuario (Id), mostrar todos los libros que ha puntuado (Título) y su puntuación.
2. Dado un usuario (Id), le recomiende cinco libros que no haya puntuado (Título). Usando un sistema de recomendación de tipo filtrado colaborativo basado en ítems.
3. Dado un usuario (Id), le recomiende cinco libros que no haya puntuado (Título). Usando un sistema de recomendación de tipo filtrado colaborativo basado en usuarios.
4. Dado un libro (Id), mostrar tres usuarios a los que se le recomendaría.
5. Dada un libro(Id), mostrar los tres libros (ISBN, título y autor) mejor puntuados.
